# POO-TPFinal

## Instalación

En el playground de Pharo ejectuar

```
Metacello new
	githubUser: 'sebaiova' project: 'POO-TPFinal' commitish: 'master' path: 'src';
	baseline: 'SI';
	load
```

## Ejecución

Con sistema vacío
```
SISistemaBibliotecas run 
```
Con datos precargados
```
SISistemaBibliotecas runPreloaded
```
